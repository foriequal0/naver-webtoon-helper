네이버 웹툰 헬퍼
=============

## 기능

<img src="./docs/weekdayList.png" width=250> <img src="./docs/weekday.png" width=250> <img src="./docs/list.png" width=300>

* 가장 마지막에 본 화 다음이 최신화면 목록을 건너뛰고 바로 최신화로 감.
* 봤던 화는 반투명하게.
* 나올 시간이 됐는데 아직 안나왔으면 나올때까지 새로고침.
* 전체 웹툰 목록을 정렬.
  * 맨날 보는거.
  * 새로 나온거.
  * 안보는건 반투명하게.
